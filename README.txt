После небольших улучшений и изменений устаревшей версии Dota 2 Building Helper, готовый представить шаблон для вашей Tower Defence кастомки!
В нём есть вся база TD начиная с постройки и заканчивая улучшениями зданий.
Ко всем кастомным и непонятным моментам мы добавили описание, но также есть вики оригинальной Dota 2 Building Helper (https://github.com/MNoya/BuildingHelper)

Установка:
1.Проходите по пути steamapps\common\dota 2 beta\game\dota_addons\НАЗВАНИЕ ВАШЕГО АДДОНА
2.Перекидывайте все файлы из паки game в ваш аддон с заменой
3.Проходите по пути steamapps\common\dota 2 beta\content\dota_addons\НАЗВАНИЕ ВАШЕГО АДДОНА
4.Перекидывайте все файлы из паки content в ваш аддон с заменой
5.и ВСЁ!

После установки вы можете делать, что угодно. Спасибо, что воспользовались нашей библиотекой!

Состав шаблона:
1)Карта на которой есть место где нельзя строить
2)Абилка на строительство
3)Абилка на продажу
4)Абилка улучшения
5)Шаблон для ваших зданий
